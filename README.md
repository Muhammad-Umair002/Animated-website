# 🌐 Animated Interactive Website

A modern, animation-rich website built to deliver a smooth and engaging user experience. This project showcases advanced front-end development using JavaScript and GSAP animations.

---

## 🚀 Overview

This website is designed as a visually appealing and interactive platform featuring smooth scrolling, dynamic transitions, and responsive layouts. It demonstrates practical implementation of animation libraries in real-world UI design.

---

## 🛠️ Tech Stack

* **HTML5** – Semantic structure
* **CSS3** – Styling, layout, responsiveness
* **JavaScript (ES6)** – Logic & interactivity
* **GSAP (GreenSock)** – High-performance animations

---

## ✨ Key Features

* 🎬 Smooth scrolling animations
* 🔄 Page transitions and motion effects
* 🖱️ Interactive UI elements (hover, click effects)
* 📱 Fully responsive design
* ⚡ Optimized performance and clean code

---

## 🎯 Sections Included

* 🏠 Hero Section with animated intro
* 📖 About Section
* 🧩 Features / Services Section
* 🖼️ Gallery / Projects Showcase
* 📞 Contact Section
